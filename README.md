# 🔬 K-means 클러스터링: 식품 영양소 분석 대시보드

학교 급식 식품의 에너지(kcal)와 나트륨(mg) 데이터를 K-means 클러스터링으로 분석하고 시각화한 인터랙티브 웹 대시보드입니다.

## 📊 프로젝트 개요

이 프로젝트는 학교 급식 식품 데이터를 분석하여 영양소 패턴에 따라 자동으로 그룹화하고, 사용자가 쉽게 탐색할 수 있는 대시보드를 제공합니다.

### 주요 기능

- 📈 **인터랙티브 산점도**: Plotly를 활용한 동적 시각화
- 🔍 **실시간 검색**: 식품명 또는 카테고리로 즉시 검색
- 🎯 **클러스터 필터링**: 특정 클러스터만 선택하여 보기
- ⭐ **음식 하이라이트**: 검색 결과에서 음식 선택 시 그래프에서 강조 표시
- 📊 **클러스터별 통계**: 각 그룹의 평균값과 주요 식품 카테고리 제공

## 🧮 분석 방법

### 데이터 전처리
- **제외 항목**: 김치류, 장아찌·절임류, 장류/양념류
- **분석 대상**: 607개 식품
- **특성 변수**: 에너지(kcal), 나트륨(mg)

### 클러스터링 알고리즘
```python
# StandardScaler를 사용한 정규화
X = df_filtered[['에너지(kcal)', '나트륨(mg)']]
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# K-means 클러스터링 (k=3)
kmeans = KMeans(n_clusters=3, random_state=42)
df_filtered['cluster'] = kmeans.fit_predict(X_scaled)
```

### 클러스터링 품질
- **실루엣 스코어**: 0.4181
  - -1 ~ 1 범위에서 0.4는 준수한 클러스터링 품질을 의미
  - 양수 값으로 클러스터가 적절히 분리됨을 나타냄
